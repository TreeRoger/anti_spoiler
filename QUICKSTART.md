# Quick Start Guide

## Installation (5 minutes)

### Chrome/Edge:
1. Open `chrome://extensions/` (or `edge://extensions/`)
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked**
4. Select the `anti_spoiler` folder
5. Done! üéâ

### Firefox:
1. Open `about:debugging`
2. Click **This Firefox**
3. Click **Load Temporary Add-on**
4. Select `manifest.json` from this folder
5. Done! üéâ

## First Use

1. **Click the extension icon** in your browser toolbar
2. **Add a show** you're watching (e.g., "Game of Thrones")
3. **Browse normally** - the extension will warn you about potential spoilers!

## Tips

- **Add custom keywords** in Settings for better detection
- **Adjust sensitivity** if you get too many/few warnings
- **Export your settings** to backup your show list

## Troubleshooting

**Extension not working?**
- Make sure it's enabled (check the toggle in popup)
- Refresh the page you're on
- Check if you've added any shows

**Too many warnings?**
- Lower the sensitivity in Settings
- Remove shows you've finished watching

**Not detecting spoilers?**
- Add custom keywords for the show
- Increase sensitivity in Settings
- Check if the show name is spelled correctly
